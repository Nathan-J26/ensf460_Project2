## PYTHON CODE INSTRUCTIONS

1. Connect the UART to your computer
2. Check which COM port the UART is using
3. Replace the PORT variable at the start of the program with the COM port being used
4. Start the python program
5. Upon pressing PB3, the program output will toggle On/Off. Python will automatically detect once the data flow has started
6. Record data for the length of time specified. Default is 60s, but can be changed by replacing the DURATION variable at the start of the program
7. Once data has been recorded, a graph of the ADC readings along with the LED Intensity will automatically appear
